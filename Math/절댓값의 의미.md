# 절댓값

-   $|5|$ 는 0 으로 부터의 5만큼의 거리를 말한다. 0 ----> 5
-   $|-5|$ 는 0 으로 부터 -5만큼의 거리를 말한다. -5 <----- 0
-   $|-5|$ 는 5 이다. 0으로 부터의 거리는 단지 5이다.

<br>

> $1<x$ 일때, $|1-x|$ 을 간소화 하시오.
-   1은 x보다 작은수 이다. 상식적으로 1에서 x를 빼면 음수 이다.
-   따라서 $1-x$은 음수 이기 때문에 절댓값 기호를 제거하게 된다면 $-(1-x) = -1+x$ 이 된다.

<br>

> $2<x<3$ 일때, $\sqrt{(2-x)^2}+\sqrt{(x-3)^2}$ 를 간소화 하시오.
- $\sqrt{x^2} = |x|$ 가 성립한다.
- 위 사항을 활용하여 간소화를 한다면?
- 왼쪽 항은 $2<x$ 이므로 $\sqrt{^2}$ 를 벗겨내면 $-2+x$ 이고
- 오른쪽 항은 $x<3$ 이므로 $\sqrt{^2}$ 를 벗겨내면 $-x+3$ 이므로 

- 식은 이와 같다. $-2+x-x+3$
- 따라서 정답은 1 이다.
